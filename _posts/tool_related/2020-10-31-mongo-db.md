# NoSQL (Not Only SQL) - Document Model

## Collection
- Mysql의 테이블처럼 동작 (Document를 묶는 단위)

## Document
- Mysql의 tuple에 해당 (attribute는 없는 형태) 
- BSON (Binary JSON) 형태로 저장된다.


## mongoose
- ODM (Object Document Mapping) : mysql의 sequelize 같은 ORM에 대응
- node.js npm 이용해 설치 가능
