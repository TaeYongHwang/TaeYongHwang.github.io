# 20.11.09 시작

# JAVA
- JDK: Java Development Kit
- JRE: Java Runtime Environment
- JVM: Java Virtual Machine

## 변수 영역
1. 클래스 멤버 변수 : static 영역 (JVM 종료 시까지 존재)
2. 지역 변수 : stack 영역 (해당 스택 영역 종료 시 삭제)
3. 객체 멤버 변수 : heap 영역 (객체와 함께 가비지 컬렉터에 의해 정리됨)
