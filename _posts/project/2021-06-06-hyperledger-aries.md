# Hyperledger Aries

## 개념
- __public key__ : verkey
- __private key__ : signing key
- __wallet__ : DIDs, Credentials, metadata 등 SSI 위한 데이터 저장
- __agent__ : 다른 entity와 상호작용하는 소프트웨어를 의미 
    - 대부분의 agent는 DID, key, VC를 안전하게 관리하기 위한 secure storage 존재.
